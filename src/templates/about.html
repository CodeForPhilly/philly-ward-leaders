About the Project

This project was built in March 2015 at a weekend hackathon called Apps for Democracy. It's not affiliated with any political party or faction; rather, its purpose is simply transparency.

What's Next

We'd love to add more data and features to this site. We've thought of a few ideas but we'd love to hear yours. Tell us!

Contributing

There's always room on the team. We can use help in several ways
  • Research - Searching google for photos, contact info, news stories, and anything else interesting, and then adding them to a spreadsheet
  • Writing - How can we best explain the ward and committee system to the general public?
  • Data Analysis - There's a lot of data out there. Some of it just needs some basic extracting/analysis to be added to the app.
  • Design - How can we improve the user experience of the site?
  • Development - Adding features and functionality requires front-end coding
Join us at the weekly Code for Philly meetup, dive into the source code, or join the conversation around pending issues.
